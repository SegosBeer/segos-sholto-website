# Allgemeine Bilder

Hier kommen alle globalen Bilder rein, die auf mehreren Seiten verwendet werden.

## Welche Bilder gehören hier hin?

| Dateiname          | Verwendung                          | Empfohlene Größe       |
|--------------------|-------------------------------------|------------------------|
| `favicon.png`      | Browser-Tab Icon (oben links)       | 64 × 64 px             |
| `og-image.jpg`     | Vorschaubild bei Social Media Links | 1200 × 630 px          |
| `logo.png`         | Logo (falls als Bild verwendet)     | beliebig, transparent  |

## Wichtige Regeln
- Dateinamen NICHT ändern (sonst funktioniert die Website nicht mehr)
- `favicon.png` → muss PNG sein
- `og-image.jpg` → muss JPG sein
- Bilder so klein wie möglich halten (max. 500 KB pro Bild)
